Welcome to Report’s documentation!

Report 0.1.0

Report is a evaluation tool that provides the following features:

  • Provides common off-the-shelf evaluation metrics including GMV impact, NTL accuracy and coverage, etc.

  • Supports both rules with fixed decision as well as parameterized rules

  • Evaluates hundreds of rules on order-level data efficiently

  • Easy to add new metrics
