Report - A Tool for Strategy Performance Evaluation and Report

Report is an evaluation tool that provides the following features:

  • Provides common off-the-shelf evaluation metrics including GMV impact, NTL accuracy and coverage, etc.

  • Supports evaluation of rules with fixed decision as well as parameterized rules

  • Evaluates hundreds of rules at the same time on order-level data efficiently

  • Easy to add new metrics

This README aims to provide a general introduction to Report.

Installation

Follow the steps below to install Report.

  1. It is recommended to create a virtual environment using virtualenv so that problems of dependencies and versions could be better addressed.

  2. Clone the repository from the remote

    git clone https://github.com/sparkle-apt/report.git

  3. Change to the Report directory

    cd report

  4. Install dependencies required by Report if they do not exist locally

    pip install -r requirements.txt

  5. Install Report

    python setup.py install --use

  6. Testing

    Try importing Report package in your local Python environment:

    >>> import report

    If you see the following error appear,

    Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
ModuleNotFoundError: No module named 'report'

    it is likely that the installation is not correct.

    Otherwise, congratulations! You have successfully installed and configured Report.

The installation duration depends on network condition, computer configuration, as well as whether dependencies are fulfilled or not. If the dependencies have been already fulfilled, the installation should take only several minutes.

Voilà!
