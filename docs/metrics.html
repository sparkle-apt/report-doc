Metrics

class report.metrics.BaseMetric(name='', df=None, rule_name=None, amt_name=None, weight_name=None, base_name=None, acc_denom_name=None, cov_denom_name=None, imp_denom_name=None, total=None, eps=1e-08)

Bases: object

Base class for metrics

Parameters

  • name (str) – name of metric

  • df (pandas.DataFrame) – dataframe based on which metrics are computed

  • rule_name (str) – name of rule decision column whose value is 1 if rejected and 0 otherwise; if None, nothing is declined

  • amt_name (str) – name of order amount column

  • weight_name (str) – name of sample weight column

  • base_name (str) – column name of losses, gmv, etc. to compute metrics; if None, equivalent to compute counts

  • acc_denom_name (str) – column name of denominator of accuracy

  • cov_denom_name (str) – column name of denominator of coverage

  • imp_denom_name (str) – column name of denominator of impact

  • total (float) – precomputed total can be passed in

  • eps (float) – epsilon

compute_total()

Equivalent to sum(base_name * weight_name).

compute_decline()

Equivalent to sum(rule_name * base_name * weight_name)

compute_accuracy()

Equivalent to 100. * decline / (acc_denom + eps), where decline is computed by report.metrics.BaseMetric.compute_decline(), and acc_denom is computed by sum(rule_name * acc_denom_name * weight_name)

compute_coverage()

Equivalent to 100. * decline / (cov_denom + eps), where decline is computed by report.metrics.BaseMetric.compute_decline(), and cov_denom is computed by sum(cov_denom_name * weight_name)

compute_base()

Compute basic indicators including total, decline, etc. that do not need comparison

compute_diff(other)

Equivalent to decline - other_decline, where decline and other_decline is computed by report.metrics.BaseMetric.compute_decline()

compute_impact(other)

Equivalent to 100. * diff / (imp_denom + eps), where diff is computed by report.metrics.BaseMetric.compute_diff(), and imp_denom is computed by sum(imp_denom_name * weight_name)
