evaluate(raw, amt_name, params=[{}], metric_names=[], weight_name=None, dcl_dcsn_names=[], rules={}, benchmark_name=None, eps=1e-08, n_jobs=2, extra={}, group_name=None)
Parameters

  • raw (pandas.DataFrame) – order level dataframe containing columns specified by amt_name, weight_name (optional), dcl_dcsn_names (optional), extra (optional), and other columns required by computation in rules

  • amt_name (str) – name of order amount column

  • metric_names (list) – list of metric names to compute; for non built-in metrics, need to first register by calling report.rule_reporter.RuleReporter.register_metric(); n and gmv are by default computed

  • weight_name (str) – name of sample weight column, defaults to None, where weights of orders are treated as 1

  • dcl_dcsn_names (list) – list of columns indicating decline decisions for each rule, indicating decline if value is 1 and 0 otherwise

  • rules (dict) –

    dict of rules, example of which is as follows

    def rule(data):
    dcl_dcsn = (data['merchant_id'] == 'some_merchant_id')
    dcl_dcsn |= (data['score'] >= 50)

    return dcl_dcsn

rules = {
    'model_score_rule': {
        'rule': rule
    },
    ...
}

    Its keys are rule reference names, and values are dict containing rule function returning decline decision (1 for declining and 0 otherwise)

  • benchmark_name (str) – defaults to None; if not None, it serves as benchmark against which all other rules compute performance lift

  • eps (float) – small number preventing being divided by zero issue

  • n_jobs (int) – number of parallel processes

  • extra (list) –

    extra info needed to compute metrics, for instance, to compute ntl, need to specify the column name of ntl in dataframe:

    [
    {
        'metric_name': 'ntl',
        'base_name': 'ntl_col'
    }
]
